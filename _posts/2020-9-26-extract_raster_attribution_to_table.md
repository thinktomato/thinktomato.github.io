1. 构建栅格数据属性表    
> 数据管理工具 -> 栅格 -> 栅格属性 -> 构建栅格属性表

2. 提取栅格属性表为单独文件    
> spatial analysis tools -> 区域分析 -> 以表格显示分区统计    
如果第1步没有做，这一步会报错

3. 将上一步栅格属性文件提取到excel表格
> 转换工具 -> Excel -> 表转 Excel

*Note：若有多幅栅格数据， 可右键相关工具，选择批处理。*
